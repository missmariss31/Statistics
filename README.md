left;"><br/></p><p style="padding-left: 5pt;text-indent: 0pt;text-align: left;">Variables:</p></li><li style="padding-top: 2pt;padding-left: 41pt;text-indent: -18pt;text-align: left;"><p word list)</p></li><li style="padding-top: 2pt;padding-bottom: 1pt;padding-left: 41pt;text-indent: -18pt;line-height: 119%;text-align: left;"><p style="display: inline;">A one-tailed positive t-test will determine whether the change in color (incongruent condition) will significantly increase the time taken to say the color of the letters in the word or whether there will be no significant change in time.</p><p style="padding-left: 95pt;text-indent: 0pt;text-align: left;"><span><img width="385" height="208" alt="image" src="StroopEffect/Image_001.jpg"/></span></p><p class="s4" </span>), containing congruent words, and the time it takes to name each color in the second list ( <span class="s7">μ</span><span class="s9">i </span>), containing incongruent words.</p><p We must continue to analyze the data if we hope to find if there is a significant difference. Let’s start out with a few stats and then get to the good stuff, our conclusion. Will we accept the null hypothesis or reject it?</p><p style="text-indent: 0pt;text-align: left;"><br/></p><p class="s11" style="padding-left: 5pt;text-indent: 0pt;text-align: left;">Moving right 0pt;text-align: left;"><br/></p><p style="padding-top: 4pt;padding-left: 5pt;text-indent: 0pt;line-height: 124%;text-align: left;">It looks like we have a relatively large difference in means. Let’s see what our t-critical ( <span class="s8">tc </span>) value is before we find our standard deviation of differences (s) and solve for <i>t</i>. Since we are using a one-tailed positive t test, we will only reject the Null ( <span class="s6">Ho </span>) Hypothesis if our <i>t </i>value is greater than our t-critical value with a percentage (p) greater than 0.05.</p><p style="text-indent: 0pt;text-align: left;"><br/></p><p style="padding-left: 5pt;text-indent: 0pt;line-height: 119%;text-align: left;">Looking at the t-table, we can see that with an alpha level of .05 and 23 degrees of freedom (n-1), our t-critical value will be 1.714.</p><p style="text-indent: 0pt;text-align: left;"><br/></p><p class="s17" style="padding-left: 8pt;text-indent: 0pt;text-align: left;">tc <span class="s18">= 1.714</span></p><p style="text-indent: 0pt;text-align: left;"><br/></p><h2 style="padding-top: 4pt;padding-left: 5pt;text-indent: 0pt;text-align: left;">Now, let’s find s and solve for t. We’re almost there!</h2><p style="text-indent: 0pt;text-align: left;"><br/></p><p class="s19" style="padding-left: 5pt;text-indent: 0pt;line-height: 120%;text-align: left;">We find s by taking the When we solved for t, we got 8.193 (rounded by three decimal places.) What does this tell us?</p><p style="text-indent: 0pt;text-align: left;"><br/></p><p class="s28" style="padding-left: 5pt;text-indent: 0pt;text-align: left;">Should we reject the Null? <b>YES!</b></p><p style="text-indent: 0pt;text-align: left;"><br/></p><p style="padding-left: 5pt;text-indent: 0pt;line-height: 119%;text-align: left;">We reject the hypothesis that there was no significant positive change because our t value is greater than our t-critical value.
